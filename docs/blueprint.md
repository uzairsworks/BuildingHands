# **App Name**: Building Hands

## Core Features:

- Service Showcase: Showcase plumbing, electrical, security camera, and maintenance services.
- Project Gallery: Display a gallery of completed projects to build trust and demonstrate expertise.
- Testimonial Display: Display client testimonials to showcase satisfaction and build credibility.
- Contact Form: A form to submit service requests.
- FAQ: An accordion-style component for frequently asked questions

## Style Guidelines:

- Primary color: Deep blue (#1A438A) to evoke trust and professionalism, suitable for technical services.
- Background color: Light blue (#D7E5F5), a very desaturated version of the primary.
- Accent color: Bright orange (#F08700) for CTAs and highlights to draw attention without overwhelming.
- Body text font: 'PT Sans' for clarity and readability.
- Headline Font: 'Playfair' for an elegant, high-end feel to contrast with the service-oriented PT Sans body text.
- Use clear, outline-style icons to represent each service category (plumbing, electrical, cameras).
- Clean, modern layout with generous white space for a professional and trustworthy feel.
- Smooth scroll effects and subtle fade-in animations for a polished user experience.
